# IDR calibration

Code accompanying the paper **“On the Calibration of Isotonic Distributional Regression”**.

We document an out-of-sample artifact of isotonic distributional regression (IDR), where probability 
integral transform (PIT) values concentrate on simple rational fractions with empirical frequencies 
resembling **Thomae’s function**, and explore mitigation via smoothing and interpolation.

## Paper

Biegert, Tobias, Johannes Resin, Alexander I. Jordan, and Sebastian Lerch (2026).
"On the Calibration of Isotonic Distributional Regression."
*Towards Trustworthy Predictions: Theory and Applications of Calibration for Modern AI, AISTATS 2026 Workshop*.
Available on [OpenReview](https://openreview.net/forum?id=4jQIBhBed7).
